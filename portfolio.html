My Spotify Stats

"My Spotify Stats" is a web application that provides users with personalized insights into their Spotify listening habits. With this app, users can explore their Top 10 Genres, Top 10 Songs, Top 10 Artists, and view a Genre Chart, all based on their listening history on Spotify.

The Guessing Name Game

The Guessing Name Game is an interactive web application built with ReactJS and ViteJS. It's a fun and challenging game where the user provides a clue, and the system tries to guess the name based on the given clue.
